# Getting Started with Create React App

```sh
npx create-react-app [nome-do-projeto]
```

### Instalar NPM:

```sh
npm install
```

### Instalar dependencias:

```sh
npm install axios

```
### Iniciar Servidor:

```sh
yarn start

```
